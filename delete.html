5분브리핑

데이터 삭제 요청

최종 업데이트 2026년 5월 19일
5분브리핑은 이용자의 데이터 삭제 요청을 존중하며, 요청 시 신속하게 처리해 드립니다. 본 페이지는 데이터 삭제 절차와 삭제 대상에 대해 안내합니다.

1삭제 대상 데이터

요청 시 다음 데이터가 삭제됩니다.

항목 설명
디바이스 식별자 앱 최초 실행 시 자동 생성된 UUID
FCM 푸시 토큰 알림 발송용 식별자
저장한 브리핑 항목 이용자가 저장 탭에 모아둔 모든 항목
알림 설정 상태 알림 켜짐/꺼짐 등 설정 값
참고 · 5분브리핑은 회원가입을 받지 않으므로, 이름·이메일·전화번호 등의 개인정보는 처음부터 수집하지 않습니다. 자세한 사항은 개인정보처리방침에서 확인할 수 있습니다.

2삭제 방법

  1. 디바이스 식별자 확인 앱을 열어 "마이" 탭으로 이동하면 하단에 디바이스 식별자가 표시됩니다.
    (디바이스 식별자가 표시되지 않는 경우, 식별자 없이도 이메일에 명시된 단말기 정보로 처리 가능합니다.)
  2. 삭제 요청 이메일 발송 아래 버튼을 눌러 이메일을 보내거나, 직접 foresight.company.lab@gmail.com으로 보내주세요.
    이메일에는 디바이스 식별자(또는 본인 확인 가능한 정보)와 삭제 요청 의사를 명시해 주시면 됩니다.
  3. 처리 및 회신 요청 접수 후 7일 이내에 삭제를 완료하고 처리 결과를 회신해 드립니다.
    삭제 후에는 동일 디바이스에서 앱을 재실행하면 새로운 식별자가 자동 생성됩니다.

3앱 삭제로 인한 자동 처리

이용자가 단말기에서 앱을 삭제하면 다음과 같이 처리됩니다.

  • 단말기 내 데이터: 앱 삭제와 동시에 모든 로컬 데이터(저장 캐시, 디바이스 식별자 등)가 즉시 삭제됩니다.
  • FCM 토큰: 알림 발송 시 토큰이 무효로 확인되면 순차적으로 서버에서 삭제됩니다.
  • 저장 항목 및 식별자: 서버에 남은 데이터는 마지막 활동 후 1년이 경과하면 일괄 삭제됩니다. 즉시 삭제를 원하시면 위 이메일로 요청해 주세요.

4자주 묻는 질문

Q. 앱을 삭제하면 끝나나요?

단말기 데이터는 즉시 사라지지만, 서버의 저장 항목과 식별자는 일정 기간 남을 수 있습니다. 즉시 모두 삭제하시려면 이메일로 요청해 주세요.

Q. 일부 데이터만 삭제할 수 있나요?

네, 가능합니다. 예를 들어 저장한 항목은 앱 내 "저장" 탭에서 개별 삭제할 수 있고, 알림만 끄려면 "마이 → 알림 설정"에서 즉시 비활성화할 수 있습니다.

Q. 디바이스 식별자를 모르겠어요.

이메일에 단말기 정보(예: "갤럭시 S24, 안드로이드 14")와 대략적인 마지막 사용 시기를 알려주시면 본인 확인 후 처리해 드립니다.

Q. 삭제 처리는 얼마나 걸리나요?

요청 접수 후 7일 이내에 처리하고 회신해 드립니다. 대부분 24~48시간 이내에 처리됩니다.
